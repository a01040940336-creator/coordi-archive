# Coordi Archive

> **Fashion Editorial Archive** — The Archive of Everyday Style.

패션 매거진 경험을 디지털 서비스로 재해석한 Fashion Editorial Archive입니다.

## 🔗 Live Demo

**GitHub Pages:** https://a01040940336-creator.github.io/coordi-archive/

## Tech Stack

| Category | Technology |
|----------|-----------|
| Framework | React 18 + TypeScript |
| Build Tool | Vite 5 |
| Styling | TailwindCSS 3 |
| Animation | Framer Motion |
| Routing | React Router DOM v7 |
| Data Fetching | TanStack Query v5 |
| Backend | Supabase (PostgreSQL + Auth + Storage) |
| Deployment | GitHub Pages via GitHub Actions |

## Features

- **Editorial Home** — 패션 매거진 커버 스타일 홈
- **Explore** — 에디토리얼 무한 스크롤 피드
- **Search** — 온도/계절/스타일/아이템 필터 조합 검색
- **Post Detail** — 패션 매거진 기사 스타일 상세 페이지
- **Upload Look** — 다중 이미지 업로드 + 태그
- **Profile** — 사용자 프로필 관리
- **Likes, Bookmarks, Comments** — 소셜 기능
- **Auth** — 이메일 회원가입/로그인

## Getting Started

```bash
npm install
npm run dev
```
